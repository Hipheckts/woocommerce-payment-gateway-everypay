WooCommerce payment gateway plugin for EveryPay credit card payments.

== Description ==

EveryPay is a card payment gateway service provider, enabling e-commerce merchants to collect credit and debit card online payments from their customers.

== Installation ==

EveryPay

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= What is needed to use EveryPay? =

You need to create account in EveryPay Merchant Portal - and for that you need an bank account in one of the supported banks.
